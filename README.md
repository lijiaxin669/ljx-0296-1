# 抢红包大作战 🧧

微信抢红包手速练手小游戏。屏幕上不断飘落红包和炸弹，玩家控制底部福袋左右移动接住红包、躲开炸弹。

---

## 🎮 游戏玩法

### 操作方式

| 设备 | 操作 |
|------|------|
| **键盘** | `←` / `→` 方向键 或 `A` / `D` 键 控制左右移动 |
| **触屏** | 点击屏幕左/右半区，或左右滑动 |
| **快捷** | `空格` / `Enter` 开始游戏 / 再来一局<br>`ESC` 返回主菜单 |

### 计分规则

| 项目 | 规则 |
|------|------|
| **红包面值** | 🔴 小红包：**+10 分**<br>🔴 中红包：**+30 分**<br>🟡 大红包：**+100 分** |
| **炸弹** | 💣 接到炸弹：**-1 条生命**（共 3 条命），连击清零 |
| **连击加分** | 1.5 秒内连续接到红包触发连击<br>**额外加分 = 基础分 × 连击数 × 10%**<br>例：5 连击接中红包（10 分）= 10 + 10×5×0.1 = **15 分** |
| **游戏时长** | **60 秒**，倒计时 10 秒时闪烁提醒 |
| **难度递增** | 随时间推移：<br>• 红包下落速度从 150 → 450（像素/秒）<br>• 生成间隔从 1200ms → 400ms<br>• 炸弹概率从 15% → 35% |

### 胜负判定
- 60 秒内得分越高越好
- 生命值归零（3 条命用完）或时间结束时游戏结束
- 本地自动记录最高分（Top 10）

## 🚀 启动方式

### 方式一：Docker Compose（推荐）

```bash
docker-compose up -d --build
```

访问 http://localhost:8080

### 方式二：Node.js 本地运行

```bash
# 安装依赖
npm install

# 开发模式（热更新）
npm run dev

# 构建生产版本
npm run build

# 本地预览构建结果
npm run start
```

## 📝 说明
